# handoff package

Source for the `/handoff` pi extension.

## Structure

- `src/handoff.ts` — the extension source (edit this)
- `tests/handoff.test.ts` — tests

## After making changes

Run tests, then install to the extensions folder:

```bash
npm test
npm run install:ext
```

`extensions/handoff.ts` is git-ignored — it is always a copy of `src/handoff.ts`.
